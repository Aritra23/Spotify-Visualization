Techonologies used to Anaylze and Visualize Spotify Data

For the data extraction we used Spotify daily ranking song playlist dataset from Kaggle. The data transformation and munging was performed used Pandas and Matplotlib. The Latitude and Longitude are derived from the Google Maps API and then merged with the transformed dataframes.

The dataframes were converted to csv and imported in Mongo as UTF-8 encoded format. The routes for each pages is created using the app.py and Flask App was used for rendering the data and binding with the template.

For the visualization we used JVectorMaps Plugin library for drawing the World Maps. It is scripted in Native browser technologies like Javascript, CSS, HTML, SVG and Jquery and works across all modern browsers and can be rendered in mobile as well as desktop version. The other interactive maps were created using Plotly.JS.
